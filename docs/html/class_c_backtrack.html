CBacktrack::CBacktrack ( const size_t  n)

The constructor. Reserve space for the permutation, diagonal, and back-diagonal arrays. Initialize the permutation array to the identity permutation. Initialize the diagonal and back-diagonal arrays to all true, indicating that all diagonals and back-diagonals are initially not occupied by a queen. Create a thread manager.
